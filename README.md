# 米尔斯海默视角 Skill（mearsheimer-perspective）

这是一个可直接用于 Codex 的人物思维 Skill，聚焦 John J. Mearsheimer（约翰·米尔斯海默）的国际关系分析框架。

## 功能

- 用米尔斯海默视角分析地缘政治与大战略问题
- 内置 5 个核心心智模型与 8 条决策启发式
- 包含结构化回答工作流（Agentic Protocol）
- 附带六维调研底稿（著作、对话、表达DNA、他者评价、决策记录、时间线）

## 目录结构

- `SKILL.md`：主技能文件
- `references/research/`：调研与提炼过程文档

## 使用方式

将本目录放到你的 Codex skills 目录下，例如：

```bash
~/.codex/skills/mearsheimer-perspective
```

然后重启 Codex，即可通过以下表达触发：

- 「米尔斯海默怎么看」
- 「用 Mearsheimer perspective 分析」
- 「用进攻性现实主义视角重算一次」
